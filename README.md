# ¿Qué es MakeMenu?

MakeMenu es un software diseñado para el sector de la hostelería. Permite a bares, restaurantes y cafeterías digitalizar su carta y automatizar el proceso de pedidos sin necesidad de instalar ninguna aplicación.

## Cómo funciona

1. El restaurante personaliza su carta digital desde un panel de administración
2. Se generan códigos QR únicos para cada establecimiento
3. El cliente escanea el QR con su móvil y accede a la carta en el navegador
4. Selecciona los productos y envía el pedido directamente a cocina
5. El personal recibe el pedido en tiempo real y puede marcar cada plato como preparado

## Ventajas principales

- **Sin instalación** — funciona completamente en el navegador, sin apps que descargar
- **Menos errores** — elimina malentendidos entre sala y cocina
- **Más rapidez** — suprime la toma de comandas manual
- **Multiplataforma** — funciona en cualquier dispositivo: móvil, tablet u ordenador
- **Personalizable** — cada restaurante tiene su propia carta
