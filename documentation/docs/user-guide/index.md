# Guía de usuario

Esta guía está dirigida a las personas que trabajan con la plataforma MakeMenu en el día a día: managers de restaurante, camareros y cocineros. Su objetivo es explicar cómo funciona la aplicación, qué puede hacer cada rol y cómo sacarle el máximo partido desde el primer día.

Si eres cliente de un establecimiento y has escaneado un código QR para hacer tu pedido, no necesitas esta guía la carta digital es completamente intuitiva y no requiere registro ni instalación.

## Cómo acceder a la plataforma

El acceso a MakeMenu depende del rol:

**Manager**
El registro como manager lo realiza directamente el equipo de MakeMenu de forma personalizada para cada establecimiento. Una vez creada tu cuenta, recibirás tus credenciales de acceso y podrás entrar a la plataforma desde cualquier dispositivo con navegador web.

**Camarero y Cocinero**
El registro del personal se hace a través de un enlace de invitación o código QR que el manager genera desde el apartado **Invitar equipo** dentro de su establecimiento. Al acceder a ese enlace, el camarero o cocinero puede registrarse con las credenciales que prefiera.

## Roles disponibles

MakeMenu tiene tres roles para el personal del establecimiento, más una vista especial para los clientes:

### Manager

Es el rol con mayor nivel de acceso. Puede gestionar todos los aspectos de su establecimiento: trabajadores, mesas, carta, ingredientes, productos y pedidos. Es el responsable de configurar la plataforma y mantenerla al día.

### Camarero

Tiene acceso a los pedidos activos de las mesas. Puede consultar el estado de cada pedido y marcarlo como **pagado** o **descartarlo** cuando corresponda.

### Cocinero

Recibe en tiempo real los pedidos solicitados por los clientes. Puede marcar cada pedido como **listo** una vez que está preparado, lo que notifica al resto del equipo.

### Vista cliente del establecimiento

No es un rol registrado en la plataforma. Los clientes acceden a la carta digital escaneando el código QR de su mesa, seleccionan los productos que desean y envían el pedido directamente a cocina — sin descargar ninguna app ni crear ninguna cuenta.

## Funcionalidades por establecimiento

Cada establecimiento tiene su propio espacio de gestión dentro de la plataforma. Desde él, el manager puede:

### Mesas

- Añadir mesas con su número de comensales
- Indicar si una mesa está disponible o no

### Pedidos

- Consultar el listado completo de pedidos
- Ver el estado de cada pedido
- Filtrar pedidos por antigüedad u otros criterios

### Equipo

- Invitar a camareros y cocineros mediante un enlace o código QR
- El personal invitado puede registrarse con las credenciales que prefiera

### Ingredientes

- Añadir, editar y eliminar ingredientes
- Definir cantidad, descripción y disponibilidad de cada uno

### Productos

- Añadir, editar y eliminar productos de la carta
- Configurar descripción, imagen, precio, categoría y disponibilidad
- Asociar ingredientes a cada producto
